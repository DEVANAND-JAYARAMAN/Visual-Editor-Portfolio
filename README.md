# visual-editor-portfolio
A modern, responsive multi-page portfolio website for a creative Photo &amp; Video Editor, built using HTML, CSS, and JavaScript with smooth animations and a sleek dark theme.

## Project Structure

All project files are kept in a single folder (root), including CSS and JavaScript assets.

```
about.html
contact.html
index.html
portfolio.html
services.html
testimonials.html
style.css
script.js
README.md
LICENSE
```
